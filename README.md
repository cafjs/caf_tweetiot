# Caf.js

Co-design permanent, active, stateful, reliable cloud proxies with your web app and devices.

See https://www.cafjs.com

## Tweet IoT

Watch a video of this app [here](https://youtu.be/TxEfYMWr8iM)

Example of collaborative multi-tenancy using devices.

Provides a Twitter-like platform for devices, synchronizing actions with timed bundles, controlling access with groups, and propagating bundles with a pub-sub service. The cloud backend is just 154 lines of JavaScript.

Actions change GPIO pins of a raspberry pi, producing sound or light.
